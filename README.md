# Overview - Assignment FUNiX
UI for client website shop tech. Clients can actions, such as: view, buy, add to cart.
# Technologies
- React Typescript
- Redux toolkit

[Deployment UI](https://shop-ten-theta.vercel.app/)
# Running on local
```
npm install
npm start
```
